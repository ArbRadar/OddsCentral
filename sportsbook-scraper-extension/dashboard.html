Database

PostgreSQL database running on port 54320

Access: localhost:54320 | User: postgres | Password: postgres

Database Tables

Schema

Key tables in the system

  • games - Unique games with UNIQUE constraint on game_id
  • odds - Bookmaker odds for each game
  • flagged_events - Games processed by matching engine
  • *_reference - Omenizer reference data (sports, teams, leagues)
  • *_mapping - Translation mappings between systems
